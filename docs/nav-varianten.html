Navigations-Blur – Varianten

Sechs Vorschläge für die Nav-Leiste im Red-Ants-Look, jeweils über einem Hintergrundfoto. Scrolle in jeder Kachel (das Foto läuft unter der Leiste durch), um den Blur zu beurteilen. Unter jeder Variante steht das CSS. Variante 1 ist der aktuelle Zustand, Variante 2 der von dir angesprochene Blur-Verlauf. Das Foto ist ein Platzhalter: oben im CSS die Variable --bg auf eine echte Red-Ants-Spielfoto-URL setzen.

Aktuell

1 · Solid-Blur

Gleichmässiger Weichzeichner auf halbtransparentem Weiss mit feiner Trennlinie. Ruhig, neutral, gut lesbar.

2 · Blur-Verlauf (progressiv)

Der Weichzeichner ist oben stark und läuft nach unten weich aus (per Masken-Verlauf). Kein harter Rand, sehr modern – das „iOS-Gefühl“.

CSS
.nav { position: sticky; top: 0;
  background: linear-gradient(to bottom,
     rgba(255,255,255,.6), rgba(255,255,255,0));
}
.nav::before {           /* geblurte Ebene mit Verlaufsmaske */
  content: ""; position: absolute; inset: 0 0 -22px 0;
  z-index: -1; pointer-events: none;
  backdrop-filter: blur(14px);
  mask-image: linear-gradient(to bottom, #000 45%, transparent);
}
